# UniBite

> Φοιτητική πλατφόρμα διαμοιρασμού σπιτικού φαγητού — University Food Sharing Platform

---

##  Περιγραφή

Το **UniBite** είναι μια web εφαρμογή που επιτρέπει σε φοιτητές να μοιράζονται σπιτικό φαγητό εντός του πανεπιστημίου. Ένας φοιτητής αναλαμβάνει τον ρόλο του **μάγειρα** και αναρτά αγγελίες με διαθέσιμες μερίδες, ενώ άλλοι φοιτητές ως **καταναλωτές** βλέπουν το feed, κάνουν κράτηση και αξιολογούν.

Η εφαρμογή αναπτύχθηκε στο πλαίσιο του μαθήματος **Προγραμματισμός & Συστήματα στον Παγκόσμιο Ιστό** — Τμήμα Μηχανικών Η/Υ & Πληροφορικής, Πανεπιστήμιο Πατρών.

##  Εγκατάσταση & Εκτέλεση

### Προαπαιτούμενα
- [WAMP Server](https://www.wampserver.com/) (Windows)

### Βήματα

**1. Clone repository**
```bash
cd C:\wamp64\www
git clone https://github.com/feritntousnikou/UniBite.git unibite
```

**2. Δημιουργία φακέλου upload**
```
C:\wamp64\www\unibite\upload\
```

**3. Εισαγωγή βάσης δεδομένων**
- Άνοιγμα [phpMyAdmin](http://localhost/phpmyadmin)
- Import → επιλογή αρχείου `database/unibite.sql`

**4. Εκκίνηση εφαρμογής**
```
http://localhost/unibite/public/login.html
```

##  Λειτουργίες

### Μάγειρας
- Δημιουργία / επεξεργασία / διαγραφή αγγελίας φαγητού (CRUD)
- Ορισμός μερίδων, τοποθεσίας παράδοσης, ωραρίου και αλλεργιογόνων
- Έγκριση / απόρριψη αιτημάτων κράτησης
- Σήμανση παράδοσης μερίδας (collected / not collected)
- Προβολή πόντων στο navbar

### Καταναλωτής
- Feed αγγελιών με χάρτη (Leaflet) και φίλτρα απόστασης
- Κράτηση διαθέσιμης μερίδας (απαιτεί τουλάχιστον 1 πόντο)
- Αξιολόγηση μάγειρα μετά την παραλαβή (1-5 αστέρια)
- Παρακολούθηση πόντων

### Admin
- Στατιστικά τελευταίου μήνα (μερίδες, αγγελίες, χρήστες)
- Leaderboard: Top Donor + γεύματα με υψηλότερη αξιολόγηση

##  Σύστημα Πόντων

| Ενέργεια                       | Αποτέλεσμα                   |
|--------------------------------|------------------------------|
| Νέος χρήστης                   | +5 πόντοι (εκκίνηση)         |
| Μερίδα βαθμολογήθηκε ≤ 3/5     | +1 πόντος στον μάγειρα       |
| Μερίδα βαθμολογήθηκε > 3/5     | +2 πόντοι στον μάγειρα       |
| Κρατημένη μερίδα δεν παρελήφθη | -1 πόντος από τον καταναλωτή |
| Δεν αξιολογήθηκε εντός 48 ωρών | -1 πόντος από τον καταναλωτή |

---

##  Κατάσταση Αγγελίας

Η κατάσταση υπολογίζεται δυναμικά στην PHP — δεν αποθηκεύεται στη βάση:

| Κατάσταση| Συνθήκη                                                           |
|----------|-------------------------------------------------------------------|
| Ενεργή   | < 48 ώρες από δημιουργία ΚΑΙ διαθέσιμες μερίδες > 0               |
| Ανενεργή | < 48 ώρες από δημιουργία ΚΑΙ διαθέσιμες μερίδες = 0               |
| Ληγμένη  | ≥ 48 ώρες από δημιουργία (κρύβεται από το feed, παραμένει στη ΒΔ) |
